# Exousia & Co. Free Upgrades Batch

Included free upgrades:

- Rewards / loyalty page: `/rewards`
- Reviews with optional photo URL: `/reviews`
- Admin abandoned cart reminder dashboard: `/admin/abandoned-carts`
- Admin CSV sales report export: `/admin/reports`
- Branded invoice system preserved at `/invoice/[id]`
- Local abandoned-cart snapshot logging from cart activity
- SQL file for optional Supabase support: `supabase/free_business_features_upgrade.sql`

No paid third-party service is required for this batch.

After uploading to GitHub and redeploying Vercel, optionally run the SQL file in Supabase SQL Editor.
