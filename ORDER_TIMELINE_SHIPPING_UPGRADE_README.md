# Order Timeline + Shipping Tracking Upgrade

Included:
- Customer-facing order timeline page at `/track`
- Admin courier name, tracking number, estimated delivery, and delivery notes fields
- Customer tracking page shortcut from admin orders
- Clear order milestones: placed, payment approved, preparing, shipped, delivered
- Cancelled/rejected order handling

If Supabase shows missing-column errors, run:

`supabase/order_timeline_shipping_upgrade.sql`
